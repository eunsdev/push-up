# Push Up, React Native Lightweight OTA Module
> [!WARNING]
> Push Up is only supported on Expo SDK 52 or later, using the Expo Bare Workflow.
## How to install
Install Push Up
```
npm install @eunsdev/push-up
```
Add plugins on app.json
```
"plugins": [
    [
        "@eunsdev/push-up",
        {
            "pushupHost": {host_server_url}
        }
    ]
]
```
Pre-build project
```
npx expo prebuild
```
